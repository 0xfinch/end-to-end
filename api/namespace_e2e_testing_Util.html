addBenchmark(benchmarks, fun, label, opt_async)code »

Adds a benchmark function to performance test array. Benchmark function will be executed repeatedly and the execution time will be measured. Return values and errors thrown from the function are ignored.

Parameters

Array to add the benchmark to.

funfunction(): goog.async.Deferred

Benchmark test to add. The function execution time will be measured. Iff opt_async is true, the function should return a deferred and execute deferred callback() when the test is complete.

labelstring

Benchmark test label

opt_asyncboolean=

Iff true, the test is asynchronous.

runPerfTests(benchmarks, opt_numSamples, opt_timeoutInterval)code »

Runs performance tests.

The argument to this method should be an array of benchmark tests. Each element of the array is of the form {benchmark: , label:name}

Parameters

An array consisting of things we want to time, and the name to show for it in the performance table.

opt_numSamplesnumber=

Number of times to run the test functions. Defaults to 10.
