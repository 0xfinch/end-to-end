e2e.openpgp.Context

OpenPGP Context implementation, that relays all operations to a ContextService available over a MessagePort. Use to create a WebWorker and unblock the UI thread when using OpenPGP Context.

new WorkerContextImpl(port)

Parameters

Port to use to communicate with the ContextService.
