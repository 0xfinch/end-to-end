class BigPrimeNum

e2e.BigNum
  └ e2e.BigNumModulus
      └ e2e.BigPrimeNum

BigPrimeNums are odd prime BigNumModuluses that could be use as the modulus in modular arithmetic operations in crypto schemes such as ECDSA or ECDH.
