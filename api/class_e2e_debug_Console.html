class Console

A Console is a lightweight replacement for goog.debug.Logger, used to record interesting events and information.

Do not use this constructor directly, instead use e2e.debug.Console.getConsole to create a console instance.

Use the debug, info, warn and error methods in the console instance similar to how you might use the same methods in the Console API. Console instances rebind these methods to the actual Console API when possible (and when requested by the e2e.debug.Console.setLevel() method.) Otherwise, they are bound to goog.nullFunction.

You may replace this behavior by implementing an instance of e2e.debug.ConsoleHandler and passing it to e2e.debug.Console.setConsoleHandler() Methods from this ConsoleHandler will then be bound to the console instances, rather than from the Console API.
