class PgpMail

Constructs an object that represents a PGP/MIME email.

new PgpMail(content, opt_preamble)

Parameters
content{attachments: ?(Array<e2e.openpgp.pgpmime.types.Attachment>), body: string, from: (string|undefined), inReplyTo: (string|undefined), subject: (string|undefined), to: (string|undefined)}

The content of the email, including necessary email headers. If an ordinary MIME message is being constructed, the content will be unencrytped. However, if a PGP/MIME message is being constructed, the the content will include encrypted data.

opt_preamblestring=

Text preceding the body of the message. This text will be appended to the beginning of the PGP/MIME message, within the root node.
