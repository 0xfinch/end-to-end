class BigNumModulus

e2e.BigNum
  └ e2e.BigNumModulus

BigNumModuluses are odd BigNums that could be used as the modulus in modular arithmetic operations in crypto schemes such as RSA.

new BigNumModulus(modulus, opt_RR)

Parameters
modulus(Array<number>|Uint8Array)

The value of the BigNum in big endian.

opt_RRArray<number>=

The precomputed Montgomery constant. RR is usually (1 << 2|N|) mod N, where N is the modulus and |N| is its length in bits. Usually |N| is the key size, but this library actually stores N as a ((key_size / 8 + 2) / 3 * 24)-bit number.
