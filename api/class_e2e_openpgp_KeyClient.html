class KeyClient

Implements a key client that searches, imports and verifies keys from/to http key server. This class implements the client side according to the following RFC: http://tools.ietf.org/html/draft-shaw-openpgp-hkp-00. Furthermore, it also verifies the consistency of the blob key's proof from the http key server, using blob transparency architecture.
