Converts a number of length 2*modulusLength_ 32-bit words into an equivalent value, mod this.modulus_, whose length is modulusLength_. The elements in the result are not guaranteed to be in the range 0 <= x < 2^32, but will definitely be in the range -10 * 2^32 <= x < 10 * 10^32.
