class SurrogateSecretKey

e2e.openpgp.packet.SecretKeyInterface

Surrogate Secret Key Packet that has no key material, and can be used in place of e2e.openpgp.packet.SecretKey for the sole purpose of signing/decryption using the provided callbacks.
