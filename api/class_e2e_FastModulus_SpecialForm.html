calculateQuotient(dividend, deltaPlusLength)code »

Returns a tentative quotient digit. The correct quotient digit is either the value returned by this method, or one less than the value returned.

When this method is called, the loop invariant guarantees that dividend / (divisor * base ^ delta) < base [base = 2^24] and the correct quotient digit we seek is q = floor(dividend / (divisor * base ^ delta)) This method returns either q or q + 1, and always returns a value less than base.
