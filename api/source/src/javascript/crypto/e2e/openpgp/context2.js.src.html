149/**
150 * Marks the key(s) as trusted for a given email address. These keys will be
151 * then returned in getKeys() calls for that user and the selected purpose(s).
152 * Other keys for that email address are implicitly marked as not trusted.
153 * The policy for trusting the keys is implemented in the KeyManager of the
154 * application.
155 * Calling this method might cause UI interaction that lets the KeyProvider
156 * manage the trusting process for the keys (e.g. entering a PIN code,
157 * confirming an action).
158 * The application may optionally pass additional data consumed by the
159 * KeyManager / KeyProvider that contain parameters for trusting keys.
160 * @param {!e2e.openpgp.Keys} keys The keys to trust.
161 * @param {!e2e.openpgp.UserEmail} email The Email address.
162 * @param {!e2e.openpgp.KeyPurposeType} purpose The purpose for which
163 * to trust the keys. Invalid purpose for a given key will be ignored.
164 * @param {e2e.openpgp.KeyTrustData=} opt_trustData Extra key trust data
165 * containing information for the KeyManager/KeyProviders.
166 * @return {!e2e.openpgp.KeysPromise} The keys.
167 * @export
168 */
