146/**
147 * Marks the key(s) as trusted for a given email address. These keys will be
148 * then returned in getTrustedByEmailKeys() calls for that user and the selected
149 * purpose(s).
150 *
151 * Other keys for that email address are implicitly marked as not trusted.
152 * The policy for trusting the keys is implemented in the PublicKeyProvider of
153 * the application.
154 *
155 * Calling this method might cause UI interaction that lets the KeyProvider
156 * manage the trusting process for the keys (e.g. entering a PIN code,
157 * confirming an action).
158 *
159 * The application may optionally pass additional data consumed by the
160 * PublicKeyProvider / KeyProvider that contain parameters for trusting keys.
161 * @param {!e2e.openpgp.Keys} keys The keys to trust.
162 * @param {!e2e.openpgp.UserEmail} email The Email address.
163 * @param {!e2e.openpgp.KeyPurposeType} purpose The purpose for which
164 * to trust the keys. Invalid purpose for a given key will be ignored.
165 * @param {e2e.openpgp.KeyTrustData=} opt_trustData Extra key trust data
166 * containing information for the PublicKeyProvider/KeyProviders.
167 * @return {!e2e.openpgp.KeysPromise} The keys.
168 * @export
169 */
170e2e.openpgp.PublicKeyProvider.prototype.trustKeys = goog.abstractMethod;
