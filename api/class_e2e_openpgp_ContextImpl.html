e2e.openpgp.Context

Implements a "context". Provides a high level API for encryption and signing services. This context is used by external code, such as the extension's user interface, to call the base OpenPGP library.

new ContextImpl(opt_keyRingStorageMechanism)

Parameters
opt_keyRingStorageMechanism?goog.storage.mechanism.Mechanism=

mechanism for storing keyring data. Defaults to HTML5 local storage.
