trustKeys(keys, email, purpose, opt_trustData)code »

Marks the key(s) as trusted for a given email address. These keys will be then returned in getTrustedByEmailKeys() calls for that user and the selected purpose(s).

Other keys for that email address are implicitly marked as not trusted. The policy for trusting the keys is implemented in the PublicKeyProvider of the application.

Calling this method might cause UI interaction that lets the KeyProvider manage the trusting process for the keys (e.g. entering a PIN code, confirming an action).

The application may optionally pass additional data consumed by the PublicKeyProvider / KeyProvider that contain parameters for trusting keys.

Specified by: e2e.openpgp.PublicKeyProvider
