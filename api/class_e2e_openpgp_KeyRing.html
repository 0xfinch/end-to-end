class KeyRing

Implements a key ring that exposes basic key management features such as generating, searching, importing, exporting keys, etc. The key ring shall be stored in browser's local storage, and shall be encrypted if the user provides a passphrase.

new KeyRing(passphrase, storageMechanism, opt_keyServerUrl)

Parameters
passphrasestring

The passphrase used to encrypt the keyring.

storageMechanismgoog.storage.mechanism.Mechanism

persistent storage mechanism.

opt_keyServerUrlstring=

The optional http key server url. If not specified then only support key operation locally.
