class EncryptedMessage

e2e.openpgp.block.Block
  └ e2e.openpgp.block.Message
      └ e2e.openpgp.block.EncryptedMessage

Representation of an Encrypted Message block. According to RFC 4880 Section 11.3, this block is represented as: - Optional repeated Encrypted Session Key (ESK) packets -- either public or symmetric. - One Encrypted Data block. Can be either a Symmetrically Encrypted Data Packet or a Symmetrically Encrypted Integrity Protected Data Packet.
