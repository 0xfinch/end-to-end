interface FastModulus

Classes that implement FastModulus have a residue(value) method that can quickly calculate value % modulus for a specific modulus.

useForMultiplicationboolean

If true, the fastest way to calculate (a * b) % modulus is to call this.residue(a.multiply(b)). If false, Montgomery reduction is faster.

Types

FastModulus.FFFFFF

An implementation of FastModulus that handles the case in which the high 24 bits of the modulus are all 1s.

FastModulus.Ox1000000

An implementation of FastModulus that handles the case in which the high bits of the modulus are a 1 followed by 24 zeros.

FastModulus.SpecialForm

This class is the common superclass of FastModulus.FFFFFF and FastModulus.Ox1000000, which have almost identical residue() methods.
